# 福井県観光調査オープンデータ

福井県観光連盟（[Fukui Tourism Federation](https://www.fuku-e.com/)）が公開し、[Code4Fukui](https://github.com/code4fukui)がホストする、福井県観光データ分析システム（FTAS）のオープンデータです。

---

## 概要

本リポジトリでは、**FTAS**（福井県観光データ分析システム）から取得した機械可読なオープンデータを提供しています。このシステムは、福井県内の観光地に設置されたQRコードベースのアンケート端末を通じて回答を収集しています。

生のアンケートデータはSFTPサーバーから毎日取得され、Denoスクリプトによって処理された後、自動化されたGitHub Actionsのワークフローを通じて本リポジトリにコミットされます。

データは福井県内の**17市町**にまたがる約**98か所の観光エリア**を網羅しており、永平寺、福井県立恐竜博物館、東尋坊、あわら温泉などの主要な観光地が含まれています。

## データファイル

データはCSVファイル形式で提供されており、直接アクセスまたはダウンロードが可能です。

### アンケート回答データ

| ファイル | 説明 |
|---|---|
| [`all.csv`](all.csv) | 全アンケート回答データ（1行につき1回答者）。毎日更新されます。 |
| [`all-cnt.csv`](all-cnt.csv) | 回答数の日次集計データ（`day`、`count`）。 |
| [`byid/`](byid/) | 単一の観光エリア・施設ごとにフィルタリングされたアンケート回答データ。6桁のファイル名 `{id}.csv` はエリアIDに対応しています。 |
| [`byid/000.csv`](byid/000.csv) | アンケートデータのスキーマ（ヘッダー行のみ）を含むテンプレートファイル。 |

**直接ダウンロードURL（GitHub Pages経由）:**
```
https://code4fukui.github.io/fukui-kanko-survey/all.csv
https://code4fukui.github.io/fukui-kanko-survey/all-cnt.csv
https://code4fukui.github.io/fukui-kanko-survey/byid/{id}.csv
```

### エリアマスターデータ

| ファイル | 説明 |
|---|---|
| [`area.csv`](area.csv) | 現在の全観光エリアのマスターリスト（ID、エリア名、説明文、座標など）。 |
| [`area2022.csv`](area2022.csv) | 2022年度のエリアマスターのスナップショット。 |
| [`area2023.csv`](area2023.csv) | 2023年度のエリアマスターのスナップショット。 |
| [`area2024.csv`](area2024.csv) | 2024年度のエリアマスターのスナップショット。 |

**直接ダウンロードURL（GitHub Pages経由）:**
```
https://code4fukui.github.io/fukui-kanko-survey/area.csv
```

## データスキーマ

### `area.csv` — エリアマスター

| 列名（日本語） | 意味 |
|---|---|
| `id` | エリアID（連番） |
| `市町名` | 市町名 |
| `親番号` | 親番号（`byid/` のファイル名に使用される6桁のID） |
| `エリア名` | エリア名 |
| `エリア説明文` | エリア説明文 |
| `通し番号` | 内部通し番号 |
| `緯度` / `経度` | 緯度 / 経度 |
| `旧エリア名` | 旧エリア名（名称変更された場合） |

### アンケート回答ファイル（`all.csv`、`byid/*.csv`）

各行は1件のアンケート回答を表します。主要な列は以下の通りです。

*   **回答者プロフィール:** 性別、生まれ年、年代、都道府県、世帯年収。
*   **訪問詳細:** 回答日時、回答エリア、市町村、宿泊数、同行者。
*   **訪問目的:** 「温泉や露天風呂」「地元の美味しいものを食べる」「名所、旧跡の観光」「買い物、アウトレット」などの動機を示す一連のバイナリフラグ列。
*   **情報収集:** 「新聞・雑誌・ガイドブック」「インターネット・アプリ」「Instagram」「友人・知人」など、訪問者がエリアについてどのように知ったかを示す一連のバイナリフラグ列。
*   **満足度とフィードバック:** 満足度スコア、NPS（ネットプロモータースコア）、不便さに関する自由記述コメントなど。

## 利用方法

`curl` などのツールを使用するか、ブラウザで直接ダウンロードURLにアクセスすることで、ローカルでの分析用にCSVファイルをダウンロードできます。

```bash
# 完全なアンケートデータセットをダウンロード
curl -L -o all.csv https://code4fukui.github.io/fukui-kanko-survey/all.csv

# 現在のエリアマスターリストをダウンロード
curl -L -o area.csv https://code4fukui.github.io/fukui-kanko-survey/area.csv
```

データは、スプレッドシートソフトやデータ分析ツール（pandas、R、Excelなど）に読み込んで、さらに詳細な分析を行うことができます。
