# 福井県観光アンケートオープンデータ

[English](README.md)

福井県観光協会が公開し、[Code4Fukui](https://github.com/code4fukui)がホストする、福井県観光アンケートシステムからのオープンデータ。[福井観光連盟](https://www.fuku-e.com/)が提供。

---

## 概要

このリポジトリは、**FTAS**（福井県観光データ分析システム）によって収集された機械可読のオープンデータを配布しています。FTASは福井県内の観光地にQRコードベースのアンケート端末を設置し、訪問者が現地で簡易アンケートに回答できるようにしています。生の回答データはCSVファイルとして公開され、GitHub Actionsによる自動ワークフローで毎日更新されます。

データは福井県の**17市町**にまたがる**約98か所の観光エリア**（北陸・若狭海岸のReihoku地域と越前内陸のReinan地域）をカバーし、永平寺、越前恐竜博物館、東尋坊、芦原温泉など多くのランドマークを含みます。

## データスキーマ

### `area.csv` — エリアマスタ

| カラム（日本語） | 意味 |
|---|---|
| `id` | 連番のエリアID |
| `市町名` | 市町名 |
| `親番号` | 親エリア番号（6桁、`byid/`のファイル名として使用） |
| `エリア名` | エリア名 |
| `エリア説明文` | エリア説明 |
| `通し番号` | 内部通し番号 |
| `緯度` / `経度` | 緯度 / 経度 |
| `旧エリア名` | 旧エリア名（改名の場合） |

### アンケート回答ファイル（`all.csv` / `byid/`）

各行は1つの完了したアンケートを表します。主なカラムグループ:

**回答者プロファイル**
- 会員ID、登録施設、性別、生年、年齢、年齢層
- 本拠地の都道府県・市町、世帯収入層
- デバイスタイプ（UA）、回答時刻、回答月

**訪問詳細**
- 回答エリア名・エリアID、市町、地域6分類、DMO
- 総宿泊日数・県内宿泊日数
- 宿泊エリア（県内/県外）
- 観光同伴者

**訪問目的**（バイナリフラグ列）
- 宿泊でのリラックス・温泉・露天風呂・地元料理・自然観賞（桜、紅葉など）・歴史的遺跡・テーマパーク/博物館/動物園・ショッピング・祭り・イベント・スポーツ/コンサート・屋外活動（水泳、釣り、ハイキング）・都市/町歩き・体験型活動・スキーリゾート/マリンスポーツ・その他のスポーツ・ドライブ/観光・友人・家族訪問・ビジネス・その他

**情報源**（バイナリフラグ列）
- 観光フェア/商品展示会・新聞/雑誌/ガイドブック・テレビ/ラジオ・パンフレット/ポスター・DMO/観光連盟ウェブサイト・インターネット/アプリ・Twitter・Instagram・Facebook・ブログ・友人・知人・その他
